# 🎓 EmotiMeme Lab

**EmotiMeme Lab** — это интерактивная платформа-тренажер, разработанная в рамках дипломного проекта для студентов-переводчиков и лингвистов. Сайт помогает изучать стратегии перевода интернет-мемов с учётом эмоциональной адаптации, опираясь на теорию базовых эмоций Пола Экмана и теорию скопоса.

---

## 📋 О проекте

Платформа включает в себя четыре основных раздела:

### 1. 📋 Testing
Тренажер для запоминания правил оформления дипломных работ (ГОСТ / методические указания).  
- Два уровня сложности: Bachelor / Master
- Мгновенная обратная связь с GIF-анимацией
- Подсчет правильных и неправильных ответов
- Модальное окно с результатами после завершения теста

### 2. 🎭 Translation Lab
Лаборатория перевода мемов, где на практике демонстрируются стратегии перевода:
- 😢 Sadness
- 😠 Anger
- 😨 Fear
- 🤢 Disgust
- 😊 Happiness
- 😲 Surprise

Каждая эмоция сопровождается мемом, описанием оригинальной эмоции и четырьмя стратегиями перевода:
- Literal Translation
- Cultural Substitution
- Visual Anchoring
- Modulation + Skopos Shift

### 3. 🏛️ Student Showcase
Галерея работ студентов университета, создавших собственные мемы, связанные с научной деятельностью.  
Изображения кликабельны и открываются в полном размере.

### 4. 📚 Theory for Translators
Теоретический раздел с объяснением ключевых концепций:
- Skopos Theory
- Multimodality
- Ekman's Basic Emotions
- Translation strategies (с таблицей стратегий)
